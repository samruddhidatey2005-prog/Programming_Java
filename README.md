# Student Management System (Java)

## Description

This project is a console-based Student Management System developed using Java. It helps manage student records like adding, viewing, updating, and deleting student details.

## Features

* Add student
* View students
* Update student details
* Delete student
* Calculate average marks

## Technologies Used

* Java
* ArrayList
* Object-Oriented Programming

##  How to Run

1. Install Java (JDK)
2. Download the file
3. Open terminal in folder
4. Compile:
   javac StudentManagementSystem.java
5. Run:
   java StudentManagementSystem

## Concepts Used

* Classes & Objects
* Loops
* Conditionals
* ArrayList
